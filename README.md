# School Messenger

Современный школьный мессенджер с архитектурой SPA (Single Page Application) и минималистичным дизайном.

## Стек технологий

- **Frontend:** React.js + TypeScript + Vite
- **Backend:** Node.js + Express + TypeScript
- **База данных:** SQLite + Prisma ORM
- **Управление состоянием:** Zustand
- **Иконки:** Lucide React
- **Стили:** Custom CSS (Clean UI)

## Функциональность

- ✅ Регистрация и авторизация (JWT)
- ✅ Лента постов с лайками и комментариями
- ✅ Создание постов с фото и опросами
- ✅ Профиль пользователя со статистикой
- ✅ Mobile-First адаптивный дизайн
- ✅ Bottom Navigation Bar
- ✅ Fullscreen Bottom Sheet для комментариев
- ✅ SVG иконки (без эмодзи)
- ✅ Категории постов (горизонтальный скролл)
- ✅ Превью изображений
- ✅ Интерактивные опросы с процентами

## Дизайн

### Современный минимализм
- Скругленные углы (12-16px)
- Мягкие тени и градиенты
- Цветовая схема: #6366f1 (индиго) и #8b5cf6 (фиолетовый)
- Качественная типографика

### Адаптивность
- **Мобильные:** Полноэкранный интерфейс
- **Desktop:** Контент ограничен 500px по ширине, центрирован, серый фон вокруг

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка базы данных

```bash
npm run prisma:generate
npm run prisma:migrate
```

### 3. Создание папки для загрузок

```bash
mkdir uploads
```

### 4. Запуск приложения

```bash
npm run dev
```

Приложение будет доступно по адресу:
- Frontend: http://localhost:3000
- Backend: http://localhost:3001

## API Endpoints

### Auth
- POST `/api/auth/register` - Регистрация
- POST `/api/auth/login` - Вход

### Posts
- GET `/api/posts` - Получить все посты
- GET `/api/posts/user/:userId` - Посты пользователя
- POST `/api/posts` - Создать пост (multipart/form-data)
- DELETE `/api/posts/:id` - Удалить пост
- POST `/api/posts/poll/:optionId/vote` - Проголосовать

### Comments
- POST `/api/comments` - Добавить комментарий

### Likes
- POST `/api/likes` - Поставить/убрать лайк
